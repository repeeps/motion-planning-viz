# 모션 플래닝, 눈으로 보기

로봇은 "**어디로, 어떻게** 갈까"를 스스로 정해야 해요 — 장애물을 피해 목적지까지. **격자 탐색·A\*·포텐셜 필드·PRM·RRT/RRT\* 너머의 경로 계획**을 **수식 없이** 익히는 인터랙티브 학습 자료. 탐색이 퍼지고 나무가 자라고 경로가 다듬어지는 걸 직접 만지며 배웁니다. 정적 HTML+JS — 빌드 없이 브라우저로 바로 열립니다.

## 시작하기

`index.html`을 브라우저로 열거나, 로컬 서버로:

```bash
python3 -m http.server 8000
# http://127.0.0.1:8000 접속
```

처음이라면 **0번 입문**(`00-intro.html`)부터 순서대로 보세요. 바쁘면 핵심 척추: 01 → 03 → 05 → 07 → 09 → 14.

## 구성 — 0번 입문 + 15챕터

| PART | 챕터 |
|------|------|
| **1 · 공간과 문제** | 01 설정공간(C-space) · 02 길찾기 = 그래프 탐색 |
| **2 · 격자 탐색** | 03 너비우선 BFS · 04 다익스트라 · 05 A\* (휴리스틱·탐욕·weighted) |
| **3 · 격자를 넘어** | 06 격자의 한계(해상도·차원의 저주) · 07 포텐셜 필드 |
| **4 · 샘플링 기반** | 08 PRM · 09 RRT · 10 RRT\* |
| **5 · 실전** | 11 경로 다듬기 · 12 궤적 최적화 · 13 차처럼 움직이기 · 14 리플래닝 · 15 너머에는 |

각 챕터는 4스텝(직관 → 인터랙션 → 결론)으로 구성됩니다. 모든 탐색·경로·곡선은 브라우저에서 실시간 계산됩니다.

## 기술

- `shared/theme.css` — 공통 다크 테마
- `shared/lib.js` — 공통 엔진 `VZ.*` (격자 탐색 BFS/다익스트라/A\*·C-space 팽창·충돌 검사·포텐셜 필드·샘플링·shortcut 다듬기 `VZ.PLAN`, 실시간 루프 `VZ.SIM` 등)
- 의존성·빌드 단계 없음

추정(어디 있나) → **계획(어디로 — 이 트랙)** → 제어(어떻게)로 이어지는 로봇 자율주행 스택의 가운데 조각입니다.
